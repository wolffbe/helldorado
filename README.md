# Helldorado — Fix

A small launcher that makes **Helldorado** (2007) run on modern **Windows 10 & 11**.

It fixes the **"Unable to initialize video mode / Could not set screen mode"** error: the game tries
to take exclusive fullscreen at a display mode modern GPUs reject. This launcher runs it **windowed at
your screen resolution** and stretches it **borderless to fill the screen** — so it looks fullscreen but
never does the failing mode-switch. Your `Helldorado.exe` is **not modified**.

## How to install

1. Download **`HelldoradoFixed.exe`** from the [**Releases**](../../releases) page.
2. Put it in your **Helldorado game folder** (the one containing `Helldorado.exe`).
   *(It also auto-finds a default GOG or Steam install, so any folder works too.)*
3. Double-click **`HelldoradoFixed.exe`** to play — it opens borderless fullscreen at your resolution.
   - If a blue Windows box appears, click **More info → Run anyway**.
4. Optional: right-click it → **Send to → Desktop (create shortcut)** for one-click launching.

# Helldorado — Fix (Deutsch)

Ein kleines Startprogramm, das **Helldorado** (2007) auf modernem **Windows 10 & 11** lauffähig macht.

Es behebt den Fehler **"Unable to initialize video mode / Could not set screen mode"**: Das Spiel will in
den exklusiven Vollbildmodus wechseln, den moderne Grafikkarten ablehnen. Dieses Startprogramm führt das
Spiel **im Fenster mit deiner Bildschirmauflösung** aus und zieht es **randlos auf Vollbild** — sieht aus
wie Vollbild, ohne den fehlschlagenden Moduswechsel. `Helldorado.exe` wird **nicht verändert**.

## Installation

1. Lade **`HelldoradoFixed.exe`** von der [**Releases**](../../releases)-Seite herunter.
2. Lege es in deinen **Helldorado-Spielordner** (dort, wo `Helldorado.exe` liegt).
   *(Eine Standard-Installation von GOG oder Steam wird auch automatisch gefunden.)*
3. Doppelklicke **`HelldoradoFixed.exe`** zum Spielen — es startet randlos im Vollbild mit deiner Auflösung.
   - Falls ein blaues Windows-Fenster erscheint: **Weitere Informationen → Trotzdem ausführen**.
4. Optional: Rechtsklick → **Senden an → Desktop (Verknüpfung erstellen)** für den Ein-Klick-Start.
